EMPLOYEE PROFILE PLACEHOLDER IMAGE

You need to create a profile placeholder image at:
src/main/resources/images/profile_placeholder.png

QUICK OPTIONS:

Option 1: Download from placeholder service
- Visit: https://via.placeholder.com/180/C2B280/1F1F1F?text=M.S
- Save as: profile_placeholder.png
- Place in: src/main/resources/images/

Option 2: Create manually
- Create a 180x180 PNG
- Fill with stone beige (#C2B280) or gray
- Add initials "M.S" in center
- Save as: profile_placeholder.png

Option 3: Use any avatar image
- Download any avatar/profile image
- Resize to 180x180
- Save as: profile_placeholder.png

The application will look for this image and display it as a circular profile photo.
If the image doesn't exist, the app will still work but show an empty circle.
